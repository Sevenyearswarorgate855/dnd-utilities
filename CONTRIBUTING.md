# Contributing to D20 Tavern

Thanks for taking interest in the project.

## Before You Start

- Check open issues before opening a new one
- Keep changes focused and easy to review
- Prefer improvements that help real table use on mobile

## How to Contribute

1. Fork the repository
2. Create a branch for your change
3. Make your update
4. Test it in the browser
5. Open a pull request with a clear summary

## Good Contribution Areas

- New dice utilities
- Better accessibility
- UI polish and motion
- Bug fixes in roll logic
- Localization improvements
- GitHub Pages or automation setup

## Pull Request Notes

- Explain what changed
- Mention how you tested it
- Include screenshots or a short recording for UI work when possible
- Avoid unrelated cleanup in the same PR

## Code Style

- Keep the project simple and readable
- Prefer small, direct functions
- Preserve the mobile-first experience
- Keep translations in sync across supported languages
