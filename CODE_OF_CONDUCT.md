# Code of Conduct

## Our Standard

We want D20 Tavern to be a welcoming project for players, creators, and contributors of all experience levels.

Examples of positive behavior:

- Being respectful and constructive
- Assuming good intent
- Giving actionable feedback
- Helping newcomers feel comfortable contributing

Examples of unacceptable behavior:

- Harassment or personal attacks
- Discriminatory language
- Trolling or intentionally disruptive behavior
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, code, or other contributions that do not align with this code of conduct.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, and discussions around the repository.

## Reporting

If someone is behaving inappropriately, contact the project maintainer through GitHub.
